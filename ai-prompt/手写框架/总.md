### 2.2 包扫描器

扫描指定包下所有带 `@Component`、`@Service`、`@RestController` 注解的类：

java

```
public class PackageScanner {
    
    public static List<Class<?>> scan(String basePackage) {
        List<Class<?>> classes = new ArrayList<>();
        // 将包名转换为路径
        String path = basePackage.replace('.', '/');
        URL url = Thread.currentThread().getContextClassLoader().getResource(path);
        if (url == null) return classes;
        
        File dir = new File(url.getFile());
        scanDirectory(dir, basePackage, classes);
        return classes;
    }
    
    private static void scanDirectory(File dir, String packageName, List<Class<?>> classes) {
        for (File file : dir.listFiles()) {
            if (file.isDirectory()) {
                scanDirectory(file, packageName + "." + file.getName(), classes);
            } else if (file.getName().endsWith(".class")) {
                String className = packageName + "." + 
                    file.getName().substring(0, file.getName().length() - 6);
                try {
                    Class<?> clazz = Class.forName(className);
                    // 检查是否有容器管理注解
                    if (clazz.isAnnotationPresent(Component.class) ||
                        clazz.isAnnotationPresent(Service.class) ||
                        clazz.isAnnotationPresent(RestController.class)) {
                        classes.add(clazz);
                    }
                } catch (ClassNotFoundException e) {
                    e.printStackTrace();
                }
            }
        }
    }
}
```

### 2.3 Bean 工厂

负责创建 Bean 实例并完成依赖注入：

java

```
public class BeanFactory {
    // 存储所有 Bean 实例，key 为 Bean 名称，value 为实例
    private final Map<String, Object> beanMap = new ConcurrentHashMap<>();
    
    // 初始化容器
    public void init(String basePackage) {
        // 1. 扫描并实例化所有 Bean
        List<Class<?>> classes = PackageScanner.scan(basePackage);
        for (Class<?> clazz : classes) {
            String beanName = getBeanName(clazz);
            try {
                Object instance = clazz.getDeclaredConstructor().newInstance();
                beanMap.put(beanName, instance);
            } catch (Exception e) {
                e.printStackTrace();
            }
        }
        
        // 2. 完成依赖注入
        for (Object bean : beanMap.values()) {
            injectDependencies(bean);
        }
    }
    
    // 依赖注入
    private void injectDependencies(Object bean) {
        Class<?> clazz = bean.getClass();
        for (Field field : clazz.getDeclaredFields()) {
            if (field.isAnnotationPresent(Autowired.class)) {
                field.setAccessible(true);
                Class<?> fieldType = field.getType();
                Object dependency = findBeanByType(fieldType);
                try {
                    field.set(bean, dependency);
                } catch (IllegalAccessException e) {
                    e.printStackTrace();
                }
            }
        }
    }
    
    // 根据类型查找 Bean
    private Object findBeanByType(Class<?> type) {
        for (Object bean : beanMap.values()) {
            if (type.isAssignableFrom(bean.getClass())) {
                return bean;
            }
        }
        return null;
    }
    
    // 获取 Bean
    public <T> T getBean(Class<T> clazz) {
        return clazz.cast(findBeanByType(clazz));
    }
    
    public Object getBean(String name) {
        return beanMap.get(name);
    }
    
    private String getBeanName(Class<?> clazz) {
        // 优先使用注解的 value，否则使用类名首字母小写
        if (clazz.isAnnotationPresent(Service.class)) {
            String value = clazz.getAnnotation(Service.class).value();
            if (!value.isEmpty()) return value;
        }
        String simpleName = clazz.getSimpleName();
        return Character.toLowerCase(simpleName.charAt(0)) + simpleName.substring(1);
    }
}
```

### 3.2 DispatcherServlet（路由分发）

负责解析 URL，找到对应的 Controller 方法并执行：

java

```
public class DispatcherServlet {
    
    private final Map<String, HandlerMethod> handlerMap = new ConcurrentHashMap<>();
    private final BeanFactory beanFactory;
    
    public DispatcherServlet(BeanFactory beanFactory) {
        this.beanFactory = beanFactory;
        initHandlerMapping();
    }
    
    // 扫描所有 Controller，建立 URL -> 方法的映射
    private void initHandlerMapping() {
        for (Object bean : beanFactory.getAllBeans()) {
            Class<?> clazz = bean.getClass();
            if (!clazz.isAnnotationPresent(RestController.class)) continue;
            
            String basePath = "";
            if (clazz.isAnnotationPresent(RequestMapping.class)) {
                basePath = clazz.getAnnotation(RequestMapping.class).value();
            }
            
            for (Method method : clazz.getDeclaredMethods()) {
                String fullPath = basePath;
                String httpMethod = "GET";
                
                if (method.isAnnotationPresent(GetMapping.class)) {
                    fullPath += method.getAnnotation(GetMapping.class).value();
                    httpMethod = "GET";
                } else if (method.isAnnotationPresent(PostMapping.class)) {
                    fullPath += method.getAnnotation(PostMapping.class).value();
                    httpMethod = "POST";
                } else if (method.isAnnotationPresent(PutMapping.class)) {
                    fullPath += method.getAnnotation(PutMapping.class).value();
                    httpMethod = "PUT";
                } else if (method.isAnnotationPresent(DeleteMapping.class)) {
                    fullPath += method.getAnnotation(DeleteMapping.class).value();
                    httpMethod = "DELETE";
                } else {
                    continue;
                }
                
                handlerMap.put(httpMethod + ":" + fullPath, 
                    new HandlerMethod(bean, method));
            }
        }
    }
    
    // 请求分发
    public void doDispatch(HttpExchange exchange) throws IOException {
        try {
            String path = exchange.getRequestURI().getPath();
            String method = exchange.getRequestMethod();
            
            // 1. 匹配 Handler
            HandlerMethod handler = handlerMap.get(method + ":" + path);
            if (handler == null) {
                sendResponse(exchange, 404, "{\"error\":\"Not Found\"}");
                return;
            }
            
            // 2. 解析参数
            Object[] args = resolveParameters(exchange, handler.method);
            
            // 3. 调用方法
            Object result = handler.method.invoke(handler.instance, args);
            
            // 4. 返回响应
            sendResponse(exchange, 200, toJson(result));
            
        } catch (Exception e) {
            e.printStackTrace();
            sendResponse(exchange, 500, "{\"error\":\"" + e.getMessage() + "\"}");
        }
    }
    
    // 参数解析
    private Object[] resolveParameters(HttpExchange exchange, Method method) throws IOException {
        Parameter[] parameters = method.getParameters();
        Object[] args = new Object[parameters.length];
        
        String query = exchange.getRequestURI().getQuery();
        Map<String, String> queryParams = parseQueryParams(query);
        
        String path = exchange.getRequestURI().getPath();
        
        // 读取请求体（如果是 POST/PUT）
        String body = null;
        if ("POST".equals(exchange.getRequestMethod()) || 
            "PUT".equals(exchange.getRequestMethod())) {
            body = new String(exchange.getRequestBody().readAllBytes(), StandardCharsets.UTF_8);
        }
        
        for (int i = 0; i < parameters.length; i++) {
            Parameter param = parameters[i];
            Class<?> type = param.getType();
            
            if (param.isAnnotationPresent(PathVariable.class)) {
                // 从路径中提取变量（需要实现路径变量提取逻辑）
                args[i] = extractPathVariable(param);
            } else if (param.isAnnotationPresent(RequestParam.class)) {
                RequestParam rp = param.getAnnotation(RequestParam.class);
                String value = queryParams.get(rp.value());
                args[i] = convertValue(value, type);
            } else if (param.isAnnotationPresent(RequestBody.class)) {
                // JSON 反序列化（可用 Jackson 或 Gson）
                args[i] = parseJson(body, type);
            }
        }
        
        return args;
    }
    
    private void sendResponse(HttpExchange exchange, int code, String response) throws IOException {
        exchange.getResponseHeaders().set("Content-Type", "application/json; charset=utf-8");
        exchange.sendResponseHeaders(code, response.getBytes().length);
        try (OutputStream os = exchange.getResponseBody()) {
            os.write(response.getBytes());
        }
    }
}
```



一个简单的 MVC 框架核心思路就是：扫描带有特定注解的类和方法，建立 URL 到方法的映射，根据请求 URL 查找并执行相应方法。

```
                result.put(item.getFieldName(), item.getString());

                result.put(item.getFieldName(), item);
            }
        }
```

### 4.2 动态代理实现 AOP

AOP 就是通过动态代理规范化实现的，让开发者用统一方式使用动态代理。核心是 `InvocationHandler` 接口：

java

```
public class AopProxy implements InvocationHandler {
    
    private final Object target;
    private final List<AspectInfo> aspects;
    
    public AopProxy(Object target, List<AspectInfo> aspects) {
        this.target = target;
        this.aspects = aspects;
    }
    
    @Override
    public Object invoke(Object proxy, Method method, Object[] args) throws Throwable {
        // 1. 执行前置通知
        for (AspectInfo aspect : aspects) {
            if (matches(aspect, method)) {
                aspect.before(target, method, args);
            }
        }
        
        Object result = null;
        try {
            // 2. 执行目标方法
            result = method.invoke(target, args);
        } catch (Exception e) {
            // 3. 异常通知
            for (AspectInfo aspect : aspects) {
                if (matches(aspect, method)) {
                    aspect.afterThrowing(target, method, args, e);
                }
            }
            throw e;
        } finally {
            // 4. 后置通知
            for (AspectInfo aspect : aspects) {
                if (matches(aspect, method)) {
                    aspect.after(target, method, args, result);
                }
            }
        }
        
        return result;
    }
    
    // 创建代理对象
    @SuppressWarnings("unchecked")
    public static <T> T createProxy(T target, List<AspectInfo> aspects) {
        return (T) Proxy.newProxyInstance(
            target.getClass().getClassLoader(),
            target.getClass().getInterfaces(),
            new AopProxy(target, aspects)
        );
    }
}
```

```
// 自定义校验注解
@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.FIELD)
public @interface NotBlank {
    String message() default "字段不能为空";
}

@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.FIELD)
public @interface Size {
    int min() default 0;
    int max() default Integer.MAX_VALUE;
    String message() default "长度不符合要求";
}

@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.FIELD)
public @interface Pattern {
    String regexp();
    String message() default "格式不正确";
}

// 校验器
public class Validator {
    
    public static void validate(Object obj) {
        Class<?> clazz = obj.getClass();
        for (Field field : clazz.getDeclaredFields()) {
            field.setAccessible(true);
            
            try {
                Object value = field.get(obj);
                
                if (field.isAnnotationPresent(NotBlank.class)) {
                    NotBlank ann = field.getAnnotation(NotBlank.class);
                    if (value == null || value.toString().trim().isEmpty()) {
                        throw new ValidationException(ann.message());
                    }
                }
                
                if (field.isAnnotationPresent(Size.class)) {
                    Size ann = field.getAnnotation(Size.class);
                    if (value != null) {
                        int len = value.toString().length();
                        if (len < ann.min() || len > ann.max()) {
                            throw new ValidationException(ann.message());
                        }
                    }
                }
                
                if (field.isAnnotationPresent(Pattern.class)) {
                    Pattern ann = field.getAnnotation(Pattern.class);
                    if (value != null && !value.toString().matches(ann.regexp())) {
                        throw new ValidationException(ann.message());
                    }
                }
                
            } catch (IllegalAccessException e) {
                throw new RuntimeException(e);
            }
        }
    }
}
```

### 6.2 通用 Mapper 实现

使用反射解析实体类，自动生成 SQL：

java

```
public class BaseMapperImpl<T> implements BaseMapper<T> {
    
    private final Class<T> entityClass;
    private final DataSource dataSource;
    
    @SuppressWarnings("unchecked")
    public BaseMapperImpl(DataSource dataSource) {
        this.dataSource = dataSource;
        // 通过反射获取泛型类型
        Type superClass = getClass().getGenericSuperclass();
        ParameterizedType type = (ParameterizedType) superClass;
        this.entityClass = (Class<T>) type.getActualTypeArguments()[0];
    }
    
    @Override
    public int insert(T entity) {
        StringBuilder sql = new StringBuilder("INSERT INTO ");
        sql.append(getTableName()).append(" (");
        
        List<Object> params = new ArrayList<>();
        Field[] fields = entityClass.getDeclaredFields();
        
        for (Field field : fields) {
            field.setAccessible(true);
            if (shouldSkipField(field)) continue;
            
            sql.append(getColumnName(field)).append(",");
            try {
                params.add(field.get(entity));
            } catch (IllegalAccessException e) {
                e.printStackTrace();
            }
        }
        sql.deleteCharAt(sql.length() - 1);
        sql.append(") VALUES (");
        for (int i = 0; i < params.size(); i++) {
            sql.append("?,");
        }
        sql.deleteCharAt(sql.length() - 1);
        sql.append(")");
        
        return executeUpdate(sql.toString(), params.toArray());
    }
    
    // 逻辑删除：将 deleteById 改为 update 操作
    @Override
    public int deleteById(Serializable id) {
        String sql = "UPDATE " + getTableName() + 
                     " SET is_deleted = 1 WHERE id = ? AND is_deleted = 0";
        return executeUpdate(sql, id);
    }
    
    // 查询时自动过滤逻辑删除的记录
    @Override
    public List<T> selectList(Wrapper<T> wrapper) {
        String sql = "SELECT * FROM " + getTableName() + 
                     " WHERE is_deleted = 0 " + wrapper.toSql();
        return executeQuery(sql, wrapper.getParams());
    }
    
    // 自动填充：在 insert/update 时设置时间戳
    private void autoFill(T entity, boolean isInsert) {
        Field[] fields = entityClass.getDeclaredFields();
        for (Field field : fields) {
            if (field.isAnnotationPresent(TableField.class)) {
                TableField tf = field.getAnnotation(TableField.class);
                if (isInsert && tf.fill() == FieldFill.INSERT ||
                    !isInsert && tf.fill() == FieldFill.UPDATE) {
                    field.setAccessible(true);
                    try {
                        if (field.getType() == Date.class) {
                            field.set(entity, new Date());
                        }
                    } catch (IllegalAccessException e) {
                        e.printStackTrace();
                    }
                }
            }
        }
    }
    
    private int executeUpdate(String sql, Object... params) {
        try (Connection conn = dataSource.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql)) {
            for (int i = 0; i < params.length; i++) {
                ps.setObject(i + 1, params[i]);
            }
            return ps.executeUpdate();
        } catch (SQLException e) {
            throw new RuntimeException(e);
        }
    }
    
    private List<T> executeQuery(String sql, Object... params) {
        List<T> list = new ArrayList<>();
        try (Connection conn = dataSource.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql)) {
            for (int i = 0; i < params.length; i++) {
                ps.setObject(i + 1, params[i]);
            }
            ResultSet rs = ps.executeQuery();
            while (rs.next()) {
                T entity = entityClass.getDeclaredConstructor().newInstance();
                // 反射设置字段值...
                list.add(entity);
            }
        } catch (Exception e) {
            throw new RuntimeException(e);
        }
        return list;
    }
}
```

```
public class ConfigManager {
    
    private static final Properties props = new Properties();
    
    static {
        // 读取激活的 profile
        String activeProfile = System.getProperty("spring.profiles.active", "dev");
        
        // 先加载默认配置
        loadProperties("application.properties");
        // 再加载环境配置（覆盖默认）
        loadProperties("application-" + activeProfile + ".properties");
        
        // 从 .env 文件加载
        loadDotEnv();
        
        // 替换环境变量占位符 ${VAR:default}
        replacePlaceholders();
    }
    
    private static void loadProperties(String fileName) {
        try (InputStream is = ConfigManager.class.getClassLoader()
                .getResourceAsStream(fileName)) {
            if (is != null) {
                props.load(is);
            }
        } catch (IOException e) {
            // 文件不存在，忽略
        }
    }
    
    private static void replacePlaceholders() {
        for (String key : props.stringPropertyNames()) {
            String value = props.getProperty(key);
            // 匹配 ${ENV_VAR:default} 格式
            Pattern pattern = Pattern.compile("\\$\\{([^}:]+)(?::([^}]*))?\\}");
            Matcher matcher = pattern.matcher(value);
            StringBuffer sb = new StringBuffer();
            while (matcher.find()) {
                String envVar = matcher.group(1);
                String defaultValue = matcher.group(2);
                String envValue = System.getenv(envVar);
                matcher.appendReplacement(sb, envValue != null ? envValue : defaultValue);
            }
            matcher.appendTail(sb);
            props.setProperty(key, sb.toString());
        }
    }
    
    public static String get(String key) {
        return props.getProperty(key);
    }
    
    public static String get(String key, String defaultValue) {
        return props.getProperty(key, defaultValue);
    }
}
```

### 7.2 @ConfigurationProperties 模拟

```
public class ConfigurationBinder {
    
    public static <T> T bind(String prefix, Class<T> clazz) {
        try {
            T instance = clazz.getDeclaredConstructor().newInstance();
            for (Field field : clazz.getDeclaredFields()) {
                String key = prefix + "." + field.getName();
                String value = ConfigManager.get(key);
                if (value != null) {
                    field.setAccessible(true);
                    Object converted = convert(value, field.getType());
                    field.set(instance, converted);
                }
            }
            return instance;
        } catch (Exception e) {
            throw new RuntimeException(e);
        }
    }
    
    private static Object convert(String value, Class<?> type) {
        if (type == int.class || type == Integer.class) return Integer.parseInt(value);
        if (type == long.class || type == Long.class) return Long.parseLong(value);
        if (type == boolean.class || type == Boolean.class) return Boolean.parseBoolean(value);
        return value;
    }
}
```

```
    static {
        // 注册异常处理器
        handlers.put(ValidationException.class, (e) ->

            new ErrorResponse(400, e.getMessage()));
        handlers.put(Exception.class, (e) -> 
            new ErrorResponse(500, "系统内部错误"));
    }
    
    public static ErrorResponse handle(Exception e) {
        ExceptionHandler<?> handler = handlers.get(e.getClass());
        if (handler == null) {
            handler = handlers.get(Exception.class);
        }
        return handler.handle(e);
    }
    
    @FunctionalInterface
    interface ExceptionHandler<E extends Exception> {
        ErrorResponse handle(E e);
    }
}
```

```
public class TraceIdFilter implements Filter {
    
    @Override
    public void doFilter(ServletRequest request, ServletResponse response, 
                         FilterChain chain) throws IOException, ServletException {
        try {
            // 生成 TraceId
            String traceId = UUID.randomUUID().toString().replace("-", "");
            TraceContext.setTraceId(traceId);
            
            // 放入响应头
            if (response instanceof HttpServletResponse) {

        } finally {
            TraceContext.clear();
        }
    }
}
```

## 十一、幂等性校验（基于 Redis）

java

```
@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.METHOD)
public @interface Idempotent {
    long timeout() default 5000;  // 毫秒
}

public class IdempotentAspect {
    
    private final JedisPool jedisPool;
    
    public Object around(ProceedingJoinPoint joinPoint) throws Throwable {
        Method method = joinPoint.getMethod();
        if (!method.isAnnotationPresent(Idempotent.class)) {
            return joinPoint.proceed();
        }
        
        Idempotent ann = method.getAnnotation(Idempotent.class);
        String key = buildKey(joinPoint);
        
        try (Jedis jedis = jedisPool.getResource()) {
            // 使用 SET NX EX 实现分布式锁
            String result = jedis.set(key, "1", "NX", "EX", ann.timeout() / 1000);
            if (!"OK".equals(result)) {
                throw new BusinessException("请勿重复提交");
            }
            return joinPoint.proceed();
        }
    }
    
    private String buildKey(ProceedingJoinPoint joinPoint) {
        // 基于用户ID + 方法签名 + 参数生成唯一 key
        return "idempotent:" + RequestContext.getCurrentUserId() + ":" + 
               joinPoint.getSignature().toShortString();
    }
}
```

## 十二、AI 限流切面

java

```
@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.METHOD)
public @interface RateLimit {
    double permitsPerSecond() default 1.0;
}

public class RateLimitAspect {
    
    private final ConcurrentHashMap<String, RateLimiter> limiters = new ConcurrentHashMap<>();
    
    public Object around(ProceedingJoinPoint joinPoint) throws Throwable {
        Method method = joinPoint.getMethod();
        if (!method.isAnnotationPresent(RateLimit.class)) {
            return joinPoint.proceed();
        }
        
        RateLimit ann = method.getAnnotation(RateLimit.class);
        String key = method.getDeclaringClass().getName() + "." + method.getName();
        
        RateLimiter limiter = limiters.computeIfAbsent(key, 
            k -> RateLimiter.create(ann.permitsPerSecond()));
        
        if (!limiter.tryAcquire(1, TimeUnit.SECONDS)) {
            throw new BusinessException("请求过于频繁，请稍后重试");
        }
        
        return joinPoint.proceed();
    }
}
```

## 总结：功能清单对照

| 原功能                              | 替代方案                    | 核心技术                       |
| :---------------------------------- | :-------------------------- | :----------------------------- |
| @Component/@Service/@RestController | 自定义注解 + 反射扫描       | Annotation + Reflection        |
| @Autowired                          | 自定义注解 + 依赖注入       | Reflection                     |
| AOP 切面                            | JDK 动态代理                | Proxy + InvocationHandler      |
| @RestController/@GetMapping         | 自定义注解 + 路由分发       | HttpServer + DispatcherServlet |
| Validation                          | 自定义校验注解 + 反射校验   | Annotation + Reflection        |
| MyBatis-Plus BaseMapper             | 手写 ORM + JDBC             | JDBC + Reflection + 泛型       |
| 逻辑删除                            | SQL 改写（DELETE → UPDATE） | JDBC                           |
| 自动填充                            | 反射设置时间戳字段          | Reflection                     |
| 多环境配置                          | Properties 文件 + profile   | Properties + 系统属性          |
| 全局异常处理                        | 统一异常捕获 + 映射         | try-catch + Map                |
| 响应增强                            | 包装器模式                  | 包装 ApiResponse               |
| Interceptor                         | Filter                      | javax.servlet.Filter           |
| 幂等性                              | Redis 分布式锁              | Jedis + SET NX                 |
| 限流                                | Guava RateLimiter           | RateLimiter                    |

**建议实施顺序**：

1. 先实现 **IoC 容器**（BeanFactory + 包扫描 + 依赖注入），这是整个框架的基石
2. 再实现 **HTTP 服务器 + 路由分发**，让项目能跑起来
3. 然后逐个添加 **AOP**、**参数校验**、**配置管理**
4. 最后实现 **ORM 持久层**

这个重构项目工程量较大，但完成后你会对 Spring 底层原理有极其深刻的理解。加油！
